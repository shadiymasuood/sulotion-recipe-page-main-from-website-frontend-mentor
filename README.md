## Features

* Fully responsive layout (mobile & desktop).
* Clean and semantic HTML structure (`main`, `header`, `article`, `section`, `footer`).
* CSS Grid used for the nutrition table.
* CSS variables for colors and consistent theming.
* Accessible list markers and clear typography hierarchy.

---

## What I changed / Suggestions applied

While building this project, I applied the following improvements and best practices:

1. **Semantic HTML structure** to improve readability and accessibility.
2. **CSS variables** for colors to make the design easier to maintain.
3. **Responsive typography** using `clamp()` for better scaling across screen sizes.
4. **Improved spacing** between list markers and text using padding on list items.
5. **Local font loading** using `@font-face` to ensure consistent typography.
6. **Mobile-first adjustments** with media queries for small screens.
7. **Consistent naming** for classes and color variables.
